About

我是张超航，2017届物理学毕业生。

自学编程有一年多了。起初是看到知乎上萧井陌大神的编程入门指南，看完之后突然 冒出一个念头，我想学编程，然后就开始了我的漫漫取经路。

起初是从 Python 开始学起，Python 确实适合入门。我用 Python 写文字版小游戏，写写简单的爬虫，写写简单的算法。不过更重要的是 Python 让我入了编程的门，知道每一门语言的基础都是数据类型，判断，循环。在学 JavaScript 时，确实感受到 Python 基础给我带来的好处。

因为学习 Python 的后端框架 flask 让我接触到了前端，导致后端没学会却入了前端的坑。前端所见即所得的学习体验让人非常有成就感。

熟悉 HTML、CSS，了解 HTML5、CSS3 并在实际项目中应用。熟悉 JavaScript，熟悉 jQuery，了解ES6基础。

三大框架选择了 Vue.js 进行学习，这是一个简单的练习。

了解 Node 和 mongodb，这是一个简单的练习。

了解 HTTP、ajax 的基础知识。

去年开始尝试写博客，起初发表在博客园中，17年接触到 hexo 也创建了一个自己的博客，现在是写了就发到自己的 hexo 博客上，挑一部分拿得出手的放在博客园的个人主页上。
